<h1 align="center">QR Code Generator</h1>
<p align="center">
  <strong>Simple Web Based Application for Links, URLs and Text Sharing.</strong>
</p>

<br>

 A lightweight, efficient tool that converts user-provided text or URLs into high-quality QR codes instantly. Perfect for sharing links, contact info, or Wi-Fi credentials.


 ---

 ## Key Features

-  **Real-time Generation:** Convert text to QR codes in milliseconds.
-  **Custom Inputs:** Supports URLs, plain text, and special characters.
-  **Downloadable:** Save your generated QR codes for offline use.
-  **Responsive Design:** Works seamlessly on desktop and mobile.

## Setup and Installation

Follow these steps to run the project on your machine (You don't need download anything just a browser needed)

1.  **Clone the repository:**
```bash
git clone https://github.com/alokjar8866/qr-code-generator.git
```


2. **Open the project folder**
```bash
cd qrcode 
//or navigate to your folder
```

3. **Run the project**
Simply open in browser:
Just double click `qr.html` - no server needed!
